# 计算159687实时估值和折溢价

## 任务目标

计算159687的实时估值并估算当前的折溢价。

## 实现步骤

1. **获取实时净值数据**

   * 访问官方网站 `https://www.csopasset.com/sg/en/products/sg-carbon/etf.php`

   * 提取 Intra-day Estimated NAV (USD) 作为官方实时净值

2. **计算历史净值**

   * 使用用户提供的 t 日净值数据：13-Feb-2026 1.9924 -0.0244 -1.21

   * 通过减法计算 12-Feb-2026 的净值：1.9924 - (-0.0244)

3. **计算涨跌幅**

   * 用实时净值减去 12-Feb-2026 的净值

   * 计算涨跌幅百分比

4. **估算折溢价**

   * 获取 159687 的当前市场价格

   * 计算折溢价：(市场价格 - 实时净值) / 实时净值 × 100%

## 技术实现

* 使用 Python 编写脚本

* 使用 `requests` 和 `beautifulsoup4` 库进行网页数据抓取

* 输出详细的计算结果和分析

* 12-Feb-2026 净值
