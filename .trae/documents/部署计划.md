# ETF估值程序部署计划

## 目标
将159687 ETF实时估值程序部署到服务器上，通过微信小程序或服务号提供实时查看功能。

## 整体架构

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│  微信小程序/服务号  │────▶│    后端API服务    │────▶│   数据获取模块    │
│     (前端)       │◀────│   (Flask/FastAPI) │◀────│ (calculate_iopv) │
└─────────────────┘     └─────────────────┘     └─────────────────┘
                               │
                               ▼
                        ┌─────────────────┐
                        │   数据库/缓存     │
                        │  (Redis/MySQL)   │
                        └─────────────────┘
```

## 第一阶段：后端API服务开发

### 1.1 技术选型
- **Web框架**: Flask 或 FastAPI（推荐FastAPI，性能更好，自带API文档）
- **任务调度**: APScheduler（定时刷新数据）
- **缓存**: Redis（存储实时数据，减少重复请求）
- **数据库**: MySQL 或 SQLite（存储历史数据）

### 1.2 API接口设计

| 接口 | 方法 | 功能 |
|------|------|------|
| `/api/realtime` | GET | 获取实时估值数据 |
| `/api/history` | GET | 获取历史数据列表 |
| `/api/refresh` | POST | 手动刷新数据 |

### 1.3 主要文件结构
```
server/
├── app.py              # 主应用入口
├── api/
│   ├── __init__.py
│   └── routes.py       # API路由
├── services/

├── models/

│   └── schemas.py      # 数据模型

│   ├── __init__.py

```

## 第二阶段：服务器部署

### 2.1 服务器选择
- **云服务器**: 阿里云/腾讯云/华为云
- **配置建议**: 
  - CPU: 1核
  - 内存: 2GB
  - 带宽: 1Mbps
  - 系统: Ubuntu 20.04 或 CentOS 7+

### 2.2 部署步骤

1. **环境准备**
   ```bash
   # 安装Python 3.9+
   sudo apt update
   sudo apt install python3 python3-pip
   
   # 安装Redis
   sudo apt install redis-server
   
   # 安装Nginx（反向代理）
   sudo apt install nginx
   ```

2. **应用部署**
   ```bash
   # 克隆代码
   git clone [repository] /var/www/etf-iopv
   
   # 安装依赖
   cd /var/www/etf-iopv
   pip3 install -r requirements.txt
   
   # 使用Gunicorn运行
   pip3 install gunicorn
   gunicorn -w 4 -b 127.0.0.1:8000 app:app
   ```

3. **Nginx配置**
   ```nginx
   server {
       listen 80;
       server_name your-domain.com;
       
       location / {
           proxy_pass http://127.0.0.1:8000;
           proxy_set_header Host $host;
           proxy_set_header X-Real-IP $remote_addr;
       }
   }
   ```

4. **使用Supervisor管理进程**
   ```bash
   sudo apt install supervisor
   ```
   
   配置文件 `/etc/supervisor/conf.d/etf-iopv.conf`:
   ```ini
   [program:etf-iopv]
   directory=/var/www/etf-iopv
   command=gunicorn -w 4 -b 127.0.0.1:8000 app:app
   autostart=true
   autorestart=true
   user=www-data
   ```

### 2.3 HTTPS配置（微信要求）
```bash
# 使用Let's Encrypt免费证书
sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx -d your-domain.com
```

## 注意事项

1. **数据获取频率**：避免过于频繁请求外部网站，建议30秒刷新一次
2. **错误处理**：网络异常时显示缓存数据，并提示用户
3. **安全性**：API接口添加访问限制，防止恶意请求
4. **合规性**：确保数据来源合法，标注数据来源
