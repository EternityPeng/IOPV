# 159687 ETF 实时估值计算 - V1 旧版本

这是项目的第一个版本，保留了原始的实现方式。新版本已经重构为插件式架构，但此版本作为蓝本保留。

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `calculate_iopv.py` | 核心计算模块，包含所有估值计算逻辑 |
| `iopv_gui.py` | GUI界面程序 |
| `backup_methods.py` | 备用方法（包含未使用的get_official_nav方法） |
| `*.json` | 缓存文件 |
| `*.txt` | 输出的估值数据文件 |

## 🚀 使用方法

### 方法一：命令行模式
```bash
cd v1_legacy
python calculate_iopv.py
```

### 方法二：GUI模式
```bash
cd v1_legacy
python iopv_gui.py
```

## 📊 功能特点

1. **实时净值获取**
   - 主要：ICE API (`https://inav.ice.com/api/1/csop/...`)
   - 备用：CSOP官网iframe页面

2. **市场价格获取**
   - 来源：新浪财经API

3. **最新基金净值**
   - 来源：akshare库

4. **历史净值对比**
   - 来源：CSOP官网echarts图表

5. **自动刷新**
   - GUI支持30秒自动刷新

6. **数据保存**
   - 关闭GUI时自动保存数据到txt文件
